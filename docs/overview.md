# プロジェクト概要

## プロジェクト名

Golf Swing Checker

## 目的

ゴルフスイングを基礎から段階的に練習し、体得するための教材を作る。

このプロジェクトでは、スイング解析や自動診断ではなく、練習の順番、チェックポイント、合格基準を明確にすることを重視する。

## 背景

ゴルフ初心者がいきなりフルスイングを練習すると、うまく当たらない理由や、何を直せばよいのかが分かりにくい。

また、練習場でただ球を打つだけでは、手先で当てにいく癖や、力任せに振る癖がつきやすい。

そのため、スイングを小さな動きに分解し、ひとつずつ確認しながら積み上げる教材が必要である。

## 作るもの

ユーザーが決められた順番で練習ステップを進め、各ステップのチェックポイントを確認しながらスイングを作っていくカリキュラム型の教材。

## 対象ユーザー

- ゴルフ用語は一通り理解している人
- スコアが130前後で、なかなか上達しない人
- スイングを基礎から作り直したい人
- 練習の順番が分からない人
- フルスイングの前に基本動作を身につけたい人

## スコープ

- ショートアイアンを使った基礎スイング作り
- 小さな動きからフルスイングへ進む段階的な練習
- 各ステップの目的、練習方法、チェックポイント、合格基準
- ユピテルの計測器を持っている前提での練習補助

## 非スコープ

- スイング動画の解析
- AIによる自動判定
- クラブ別の打ち分け
- 飛距離アップに特化した練習
- 高度なスイング理論
- 弾道計測機器との連携
