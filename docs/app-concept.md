# アプリ構想

## アプリの役割

ユーザーが今日やる練習ステップを確認し、練習内容とチェックポイントを1か所で扱えるようにする。

## ユーザー体験

1. 現在のステップを見る
2. 練習内容を確認する
3. 練習する
4. チェックポイントを確認する
5. ミート率や飛距離などの計測値を、チェックポイントの一部として確認する
6. 次に進むか、同じステップを続けるか判断する

## 主要画面

- ホーム
- ステップ一覧
- ステップ詳細

## 主要機能

- 現在のステップ表示
- ステップ完了チェック
- チェックポイント内でのユピテル計測値の目安提示

## 計測値の扱い

ユピテルの計測値は、飛距離を伸ばすためではなく、各ステップに合った出力で練習できているかを確認するために使う。

たとえばハーフスイングでは、飛距離の目安を示すことで、力の入れすぎや振り幅の大きくなりすぎを抑制する。

ミート率、飛距離、ヘッドスピードなどの数値は、チェックポイントの一部として扱う。
